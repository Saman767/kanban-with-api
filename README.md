💻 Kanban Board With API 💖

A modern, pastel-themed Kanban Board with drag & drop functionality and live motivational suggestions to keep you productive while coding ✨

✨ Features
📝 Add and delete tasks easily
🔄 Drag & drop tasks between To Do / In Progress / Done
🔢 Task counters update automatically
💡 Live motivational suggestions using Advice API
🎨 Beautiful pastel UI with smooth hover effects
✨ GSAP animations for a modern, interactive experience

HTML5
CSS3
JavaScript (Vanilla JS)
API Integration (Advice API)
GSAP (for animations)
